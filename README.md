### install-octave-ubuntu

```sh
sudo add-apt-repository ppa:octave/stable
sudo apt-get update
sudo apt-get install octave
```

```sh
sudo apt-get build-dep octave
wget ftp://ftp.gnu.org/gnu/octave/octave-4.0.0.tar.gz
tar xf octave-4.0.0.tar.gz
cd octave-4.0.0/
./configure
make 
sudo make install
```
